I am excited to share a project that exemplifies my skills and approach to development: https://lecionario.app

Faced with challenges in implementing smooth interactions within a tab view using React Native, I realized I knew how to achieve the same outcome with CSS. Opting for SolidJS, I expanded my toolkit to deliver a solution that prioritizes performance and user experience.

This app is a PWA that embodies the principles of local-first web development, harnessing the power of service worker cache and IndexedDB to provide seamless offline support. Moreover, optimization strategies, such as bundle size reduction and dependency splitting, ensure fast loading times. By incorporating some new CSS features, like scroll snap and scroll-driven animations, it delivers a mobile-native experience that sets it apart from traditional web applications.
